# 考研手账专栏 - 组件化使用说明

## 1. 项目结构
- `总论_学习到底是什么.html`: 主入口文件，包含所有逻辑与样式。
- `daolin_texture.svg`: 背景纹理素材。

## 2. 如何新增文章
目前文章内容通过 JavaScript 变量 `markdownText` 嵌入在 HTML 底部。若要新增或切换文章，有以下几种方式：

### 方式一：直接修改 HTML (最简单)
1. 打开 `总论_学习到底是什么.html`。
2. 找到底部的 `<script>` 标签内的 `const markdownText = \`...\`;` 部分。
3. 将反引号 \` \` 之间的内容替换为新的 Markdown 文本。

### 方式二：多文章支持 (扩展建议)
如果需要支持多篇文章切换，建议改造数据结构：
```javascript
const articles = {
    'id_1': {
        title: '文章标题',
        date: '2026.03.06',
        content: \`# Markdown内容...\`
    },
    'id_2': { ... }
};

function loadArticle(id) {
    const data = articles[id];
    // 渲染逻辑...
}
```

## 3. 如何修改标签与卡片
文章列表卡片位于 HTML 的 `<aside>` 标签内：
```html
<div class="article-card active" onclick="scrollToTop()">
    <!-- 标签：修改文本与背景色 -->
    <div class="card-tag">Metacognition</div>
    
    <!-- 标题 -->
    <h3 class="card-title">总论：学习到底是什么？</h3>
    
    <!-- 元信息 -->
    <div class="card-meta">
        <span>📅 2026.03.06</span>
        <span>⏱️ 10 min read</span>
    </div>
</div>
```
- **修改标签颜色**：可以在 `.card-tag` 上添加内联样式 `style="background: #颜色值"`，或者在 CSS 中定义新的类（如 `.tag-blue { background: var(--accent-blue); }`）。
